Pagination

An element which provide pagination links for a site or app with the multi-page pagination component.

Usage

Pagination example. Pagination example with jumper. Once the cursor changed, the current index will be outputed in console(F12).

API

Prop Description Type Default Value
cursor Specifies the current page Number for the navigator. Number 1
size Specifies the max size for the navigator. Number null
jumper Specifies that the "Go to" label should be displayed . boolean not-specified
change Specifies the callback for page change event. Function not-specified
