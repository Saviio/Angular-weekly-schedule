Button

An element which will replace the native element, it is used for responding user's action.

Usage

Use the type propery to quickly create a styled button. Button group with different size.

Each button instance own a loading status.





Disabled Button



API

Prop Description Type Default Value
type Specifies the type for the buttion. Option list: primary default danger String default
size Specifies the size for the button. Option list: small normal large String normal
content Specifies the content for the button. String ''
loading Specifies that the status of button should be loading. boolean false
disabled Specifies that the button should be disabled. boolean false
control Binding of the button instance. It can be accessed from controller scope. identifier undefined

Button - control

Method Description Signature
done Switches the state from loading to done. () => void
loading Switches the state from done to loading. () => void
allow Switches the state from disable to enable. () => void
disable Switches the state from enable to disable. () => void
