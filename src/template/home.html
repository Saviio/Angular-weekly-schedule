Fermi.UI - Yet another Angular UI library

Reuseable & Nestable

Fermi-UI is a set of components that implements with AngularJS. The main purpose of this project is to make people are able to build a Angular based application quickly and conveniently.

Thus, Fermi-UI encapsulated some typical components. Each component can be used or combined flexibly and may help people to get a much stronger UI function.

Dedclarative

Most of components in the library was packaged as a custom markup element (directive).

With this approach, People will have the ability to apply them to application in a dedclarative way.

Element
< Fermi : Popover .... ></ Fermi : Popover >

In general, A Fermi component can be declared in HTML template with one line code. But we also designed some parameters and ensuring it's flexible enough to evolve with people's needs.

Progress, Popover, Notification, Button and much more were provided. Please find document for detailed API reference.

Workflow

Fermi-UI was built on top of some the trendiest web development stack like webpack, ES6 & part of ES7 features, Babel, SASS.

It will be easy to integrate with your webpack project: 


                    import 'Fermi-UI/Menu'
                    let app = angular.module('app', ['Fermi.menu'])

import styles: 


                    import 'Fermi-UI/index.css'
