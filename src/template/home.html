Fermi.UI - Yet another Angular UI library

Reuseable & Nestable

Fermi-UI is a set of components that implements with AngularJS. The main purpose of this project is to make developers are able to build a Angular based application speedly and conveniently.

Thus, Fermi-UI offered about 17 components. And each component can be used or combined flexibly in a variety of scenarios. It can help developer to enhance and build a much stronger UI function.
