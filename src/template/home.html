Fermi.UI - Yet another Angular UI library

Reuseable & Nestable

Fermi-UI is a set of components that implements with AngularJS. The main purpose of this project is to make people can build a Angular based application speedly and conveniently.

Each component can be used in a variety of scenarios and it can be nested so that user can get a stronger UI function module.
