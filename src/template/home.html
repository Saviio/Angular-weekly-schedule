Fermi.UI - Yet another Angular UI library

Reuseable & Nestable

Fermi-UI is a set of components that implements with AngularJS. The main purpose of this project is to make developers are able to build a Angular based application quickly and conveniently.

Thus, Fermi-UI offered about 17 components. And each component can be used or combined flexibly in a variety of scenarios. It can help developer to get a much stronger UI function.

Fully Dedclarative

Workflow

Fermi-UI was built on top of some the trendiest web development stack like webpack, ES6 & part of ES7 features, SASS.

It will be easy to import them to project: 


                    import 'Fermi-UI/Menu'
                    let app = angular.module('app', ['Fermi.menu'])

import styles: 


                    import 'Fermi-UI/index.css'
