Radio

An element represents a selection of one item from a list of items.

Usage

... Current Status: [{{Radio.radio.checked}}], Value: [{{Radio.radio.value}}] Current Value: [{{Radio.group.value}}]

API

Radio

Props Description Type Default Value
label Specifies the label for the radio. string ''
value Specifies the value for the radio. string ''
checked Specifies that the radio should be checked. boolean not-specified
disabled Specifies that the radio should be disabled. boolean not-specified
change Specifies the callback for value change event. function not-specified

Radio Group

Props Description Type Default Value
change Specifies the callback for value change event. if user declared this propery then the "change callback" which is declared on radio propery will be ignored. function null
control Binding of the radioGroup instance. It can be accessed from controller scope. identifier undefined
