API

Menu

Prop Description Type Default Value
mode Specifies that the menu mode is "horizontal" or "vertical". Option list: H V String V

SubMenu

Prop Description Type Default Value
title Specifies the title of submenu. String ''
actived Specifies that the menu item should be expaned. It will only be effective when the mode is vertical(V). boolean not-specified

MenuItem

Prop Description Type Default Value
disabled Specifies that the menu item should be disabled. boolean not-specified
