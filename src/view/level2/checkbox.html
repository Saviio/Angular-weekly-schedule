Checkbox

An element allows a user to select a value from two options.

Usage

Checkbox example. Current Status: [ {{Checkbox.checkbox.checked}} ], Once the value of checkbox changed, it will be outputed in console(F12).

API

Prop Description Type Default Value
label Specifies the label for the checkbox. String ''
checked Specifies that the checkbox should be checked. boolean true
disabled Specifies that the checkbox should be disabled. boolean true
change Specifies the callback for value change event. Function null
control Binding of the checkbox instance. It can be accessed from controller scope. identifier undefined

Checkbox - control

Prop Description Signature
allow Switches the state from disable to enable. () => void
disable Switches the state from enable to disable. () => void
checked Sets or retreves the state of the check box not a function
