i18n

Internationalization for some component.

Usage

Fermi-UI provides very weak i18n support for some component. e.g. Modal, Select. app.config(['FMi18nProvider', i18n => i18n.locale('zhCN')])

API

Method Description Signature
locale Specifies the lang for Fermi module, Fermi will use 'zhCN' by default. (lang: String) => void
localize Sets the new language package. (lang: String, package: object) => void
