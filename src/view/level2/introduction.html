Fermi-UI #

For developers building Angular based application, this project provides a set of reuseable UI components (AngularJS directive and CSS). It will be easy to use and integrate with webpack workflow.

Please find the document which described detail component design on the left. Thank you for your reading.

This library was inspired by excellent Project: Ant.Design. If you're a fan of React, it will be one of your best ui-component solutions.

Installation # 


             npm install Fermi-UI --save

Usage #

import single component: 


import 'Fermi-UI/Menu'
let app = angular.module('app', ['Fermi.menu'])

import styles: 


import 'Fermi-UI/index.css'

Compatibility #

IE9+

Customization #

Although the styles of compoents are defined in the source code of component, there're some way to customize the component.

Via Component Property

I designed some property for a little foreseeable scenarios. For example, the color of progress bar.

So, developer can just pass a customized property into component, then it will be apply to the result markup.

Via override CSS in the runtime

Since style overriding is a excellent feature which is supported by browser, Developers can apply a different style in the application css file with a more specific element selector.

Example: 


#container > button.#button{
    background:#3683c6;
    border:none;
}

Via Webpack Loader? (not supported yet)

Sorry, I'm still consider about the best way of customizing component, Maybe I will refactor the structure of CSS and add this part in the future.
