Notification

An element displays notification that explains latest information.

Usage

Make a default notification on the right. //inject Fermi.Notification in controller first. notification.default('Normal', "Here's a normal message."") Notification with different type. notification.normal('Normal', "Here's a normal message.") notification.success('Success', "Here's a successful message.") notification.warn('Warning', "Here's a warning message.") notification.error('Error', "Here's a error.") A customized notification. Once notification is removed, a message will be printed in console(F12). let option = { topic: 'customized message', message: 'After notification is removed, please check the output in console.', type:'normal', callback:() => console.info('callback is executed.') } notification.send(option)

API

* Note: Please inject Fermi.Notification as a service in your controller first.

Method Description Signature
default Makes a default notification. (topic: String, message: String) => void
normal Makes a normal notification. (topic: String, message: String) => void
success Makes a successful notification. (topic: String, message: String) => void
warn Makes a warning. (topic: String, message: String) => void
error Makes a error. (topic: String, message: String) => void
send The underlying method of notification service. (option: object) => void
