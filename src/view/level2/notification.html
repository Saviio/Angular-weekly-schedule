Notification

A control element used as a navigational aid in interface.

Usage

... //inject Fermi.Notification in controller first. notification.default('Normal', "Here's a normal message."") ... notification.default('Normal', "Here's a normal message.") notification.success('Success', "Here's a successful message.") notification.warn('Warning', "Here's a warning message.") notification.error('Error', "Here's a error.") ... let option = { topic: 'customized message', message: 'After notification is removed, please check the output in console.', type:'normal', callback:() => console.info('callback is executed.') } notification.send(option)

API

Note: Please inject Fermi.Notification as service in your controller first.
