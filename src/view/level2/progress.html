API

Line

Props Description Type Default Value
value Specifies the value for the linear bar. identifier undefined
default Specifies the default value for the linear bar. number 0

Circle

Props Description Type Default Value
value Specifies the value for the circle bar. identifier undefined
default Specifies the default value for the circle bar. number 0
shape Specifies the shape for the circle bar. option list: rect round string round
inner Specifies the inner color for the circle bar. string #e9e9e9
outer Specifies the outer color for the circle bar. string #488fcd
progress Specifies that the circle bar should be effect after its value changed. string not-specified
unit Specifies the unit character for circle bar info. string %
showinfo Specifies the value number will be shown in component. option list: true false string false

Query

Props Description Type Default Value
actived Specifies that the query bar should be displayed. boolean not-specified
control Binding of the query bar instance. It can be accessed from controller scope. identifier undefined

Query - control

Method Description Signature
show Calling the method to display the query bar. () => void
hide Calling the method to hide the query bar. () => void

LoadingBar Service

* Note: Please inject Fermi.Loading as service in your controller first.

Method Description Signature
start Calling the service to start the loading animation. () => void
done Calling the service to finish the loading animation. () => void
inc Increasing the loading bar percentage value. number => void
set Setting the loading bar percentage value. number => void
