Fermi.UI - Yet another Angular UI library

Reuseable & Nestable

Fermi-UI is a set of components that implements with AngularJS. The main purpose of this project is to make developers are able to build a Angular based application quickly and conveniently.

Thus, Fermi-UI encapsulated some typical components. Each component can be used or combined flexibly and may help developers to get a much stronger UI function.

Dedclarative

Most of components in the library was packaged as a custom markup element (directive).

With this approach, Developers will have the ability to apply them to the application in a dedclarative way.

Element
< Fermi : Popover .... ></ Fermi : Popover >
Element
< Fermi : Line .... ></ Fermi : Line >

Follow the guide, Just declare them in the view template like built-in html element. Also, we designed different parameters to ensure they're flexible enough to evolve with people's needs.

Progress, Popover, Steps, Button and much more components were provided in the lib. Please find document for detailed design.

Workflow

Fermi-UI was built on top of some the trendiest web development stack like Webpack, ES6 & part of ES7 features, Babel, SASS.

It will be easy to integrate with your Webpack project: 


import 'Fermi-UI/Menu'
let app = angular.module('app', ['Fermi.menu'])

import styles: 


import 'Fermi-UI/lib/index.css'
